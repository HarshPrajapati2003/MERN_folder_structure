## Frontend Setup

1. Navigate to the Frontend directory:
    ```shell
    cd Frontend
    ```
2. Install the necessary dependencies:
    ```shell
    npm install
    ```
3. Start the Frontend server:
    ```shell
    npm run dev
    ```

## Backend Setup

1. Navigate to the Backend directory:
    ```shell
    cd Backend
    ```
2. Install the necessary dependencies:
    ```shell
    npm install
    ```
3. Start the Backend server:
    ```shell
    npm run server
    ```
